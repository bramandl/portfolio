Project Date September to October, 2024
Role Team Developer

Context

As part of my training, I took part in the Web Listing Project, where I worked on both the backend and frontend of the project. The goal was to build a web platform where users could access and manage listings of various products.

My Role and Responsibilities

In this project, I took the initiative to structure and organize the development process, ensuring the smooth collaboration between the team members. On the backend side, I was responsible for setting up the entire git repository and ensuring the project followed best practices for version control. I also initialized the project base, created routes, and ensured that the database structure was ready for the other team members to start their work. On the frontend side, I helped to set up the initial structure, created components, views, and applied the necessary styling to ensure that the platform was user-friendly and clean, making it easier for my colleagues to build upon. My experience in the workplace helped me take charge of these tasks, enabling me to assist my colleagues and ensure the project progressed efficiently.

Technologies Used

  • Backend: Symfony, PHP, MySQL
  • Frontend: HTML, CSS, JavaScript
  • Version Control: GitLab

Results

  • Successful project initialization and smooth collaboration between team members.
  • Creation of a functional backend structure, with routes and database setup, ready for further development.
  • Clean and user-friendly frontend, helping the team continue their work without friction.
